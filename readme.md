# Rentool

Peer-to-peer rental services platform enabling users to rent and list items within their community.

## Overview

Rentool is a backend service that facilitates peer-to-peer rentals, connecting item owners with renters through a seamless marketplace experience.

## Features

- User authentication and profiles
- Item listing and management
- Rental booking system
- Payment processing
- Reviews and ratings
- Search and filtering

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- Database (PostgreSQL/MongoDB)
- Environment variables configured

### Installation

```bash
git clone <repository-url>
cd rentool-be
npm install
```

### Running the Server

```bash
npm start
```
